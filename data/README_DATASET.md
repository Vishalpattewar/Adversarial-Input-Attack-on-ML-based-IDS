# Dataset Setup Instructions

This folder contains dataset files for the
**Adversarial Input Attack on ML-based IDS** project.

---

## ⚠️ Important — What Is NOT in This Repository

```
data/cicids2017/cicids2017_cleaned.csv  ← NOT on GitHub (685MB)
data/network_flows_cicids.csv           ← NOT on GitHub (generated)
```

Both files are excluded from GitHub because:
- `cicids2017_cleaned.csv` = 685MB (exceeds GitHub limits)
- `network_flows_cicids.csv` = generated by `prepare_cicids.py`

Follow the steps below to set up both files.

## Step 1 — Download CICIDS2017 Cleaned Dataset

### Dataset Details

```
Name    : CICIDS2017 Cleaned and Preprocessed
Source  : Kaggle
Author  : Erica Anacleto Ribeiro
URL     : https://www.kaggle.com/datasets/
          ericanacletoribeiro/
          cicids2017-cleaned-and-preprocessed
File    : cicids2017_cleaned.csv
Size    : 685 MB
Rows    : 2,520,751
Columns : 53 (52 numeric features + Attack Type label)
```

### Why This Version

- ✅ Already cleaned — no inf values, no whitespace issues
- ✅ Single CSV file — simple loading
- ✅ 52 clean features — better than original 76 noisy features
- ✅ Usability rating 10.0 on Kaggle
- ✅ Well tested — 8,000+ downloads, 73+ notebooks

---

### Option A — Kaggle CLI (Fastest)

```bash
# Step 1: Install Kaggle CLI
pip install kaggle

# Step 2: Setup API key
# Go to: kaggle.com → Account → Settings → API → Create New Token
# Downloads kaggle.json to your Downloads folder

# Linux / WSL / macOS:
mkdir -p ~/.kaggle
cp ~/Downloads/kaggle.json ~/.kaggle/
chmod 600 ~/.kaggle/kaggle.json

# Windows:
# Copy kaggle.json to: C:\Users\YourName\.kaggle\kaggle.json

# Step 3: Download dataset
cd data/cicids2017
kaggle datasets download \
    -d ericanacletoribeiro/cicids2017-cleaned-and-preprocessed

# Step 4: Extract
unzip cicids2017-cleaned-and-preprocessed.zip
cd ../..
```

### Option B — Manual Browser Download

```
1. Open browser

2. Go to:
   https://www.kaggle.com/datasets/ericanacletoribeiro/
   cicids2017-cleaned-and-preprocessed

3. Sign in to Kaggle (free account required)

4. Click the Download button

5. Save the ZIP file

6. Extract and place file:

   Linux / WSL / macOS:
     unzip cicids2017-cleaned-and-preprocessed.zip
     mkdir -p data/cicids2017
     mv cicids2017_cleaned.csv data/cicids2017/

   Windows:
     Extract ZIP
     Copy cicids2017_cleaned.csv to:
     adversarial-ids-cicids\data\cicids2017\
```

## Step 2 — Verify Download

```bash
# Check file exists
ls -lh data/cicids2017/cicids2017_cleaned.csv
# Expected: ~685MB

# Check row count
wc -l data/cicids2017/cicids2017_cleaned.csv
# Expected: 2520752 (2,520,751 rows + 1 header)

# Check first row (column names)
head -1 data/cicids2017/cicids2017_cleaned.csv
# Expected starts with: Destination Port,Flow Duration,...

# Check last column (label)
head -1 data/cicids2017/cicids2017_cleaned.csv | \
    tr ',' '\n' | tail -1
# Expected: Attack Type

# Quick data check
python3 -c "
import pandas as pd
df = pd.read_csv('data/cicids2017/cicids2017_cleaned.csv',
                 nrows=100)
print(f'Columns  : {len(df.columns)}')
print(f'Label col: {df.columns[-1]}')
print(f'Labels   : {df[df.columns[-1]].unique()}')
"
# Expected:
# Columns  : 53
# Label col: Attack Type
# Labels   : ['Normal Traffic' 'Port Scanning' ...]
```
